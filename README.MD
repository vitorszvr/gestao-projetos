# Controle de Fluxo de Caixa Pessoal

Este projeto foi desenvolvido para a disciplina de **Gestão de Projetos**. É um sistema simplificado de gestão financeira pessoal para controle de receitas e despesas.

## 🚀 Funcionalidades (MVP)
- **Cadastro de Transações**: Inserção de descrição, valor, data e tipo (Receita/Despesa).
- **Listagem de Movimentações**: Visualização de todos os registros em tabela.
- **Resumo de Saldo**: Dashboard com cálculo automático de Total de Entradas, Total de Saídas e Saldo Líquido.
- **Exclusão**: Remoção de transações do histórico.
- **Persistência Básica**: Dados são armazenados localmente no seu navegador (`localStorage`), para que você não perca os registros ao fechar a aba.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 💻 Como executar
Este projeto roda diretamente no navegador, sem a necessidade de instalar dependências.

1. Faça o clone deste repositório ou baixe os arquivos.
2. Acesse a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador padrão.
4. Comece a adicionar suas receitas e despesas!
